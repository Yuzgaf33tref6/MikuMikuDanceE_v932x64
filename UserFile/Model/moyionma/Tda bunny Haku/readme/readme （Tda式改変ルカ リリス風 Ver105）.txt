○モデル説明
髪の毛が荒ぶって体の前面にまとわりつかない様にガード用の剛体を追加しました。
モーションによっては髪の毛が不自然な動きになるかもしれませんので、その場合はPMDエディタ等で「髪ガード」の剛体を削除して下さい。

逆光の時に顔だけ白くなるのが好きでは無かったので、本モデルでは顔にtoonを設定しています。
他のモデルと合わせて使う場合、顔の色に違いが出て不自然になるかもしれません。
その場合は、PMDエディタ等で顔のtoon設定を無くして下さい。
（材質の「顔」の項目のtoonの欄を空欄にすればＯＫです。）

そぼろ様の作成したエフェクト「AutoLuminous」に対応しています。
AutoLuminous用の光量調整モーフを仕込んでありますので、光量調整はそちらを使う方が楽かと思います。
（LightUp　LightOffで光量調整ができます。）

○モリガン風、リリス風　モデル説明
羽の物理演算を抜いたモデルは無いので、必要な方はPMDエディタ等で羽部のジョイントと剛体を消して（注意：ジョイントを先に消して下さい。）表示枠の羽のところに羽部のボーンを追加して下さい。それで羽部も手動で操作可能になります。

黒ワンピモデルは、初期状態にて髪の毛の剛体がスカートの剛体に突き刺さっています。そのまま動かすと髪の毛がスカートからなかなか抜けず、動きがおかしくなってしまう事が多いです。
最初に首を振る等して、髪の毛をスカートから抜いて動かす様にして下さい。
