
■■　MMD MotionWorks　■■

新規導入やバージョンアップの手間を省くため、
MMDのモーションに関わる自作ツール群をパッケージ化しました

現在は以下のツールが含まれます(2010/8/1)

・MMD PathMaker 0.7
　・およびそのプラグイン

・MMD CameraObject 0.1

・VMDEditor 0.2

詳細はそれぞれのReadmeを参照してください


●●　共通事項　●●

■インストール

不要です
設定ファイルを作らないので、どこに置いても特に問題ないはず
アンインストールも、まとめて削除すればOKです

■動作環境

.NET Framework 2.0
おそらくXP以降のWindows
（Vista64bitで動作確認）

■開発環境

Microsoft Visual C# 2005 Express Edition
