【名称】回転メイカー
【種別】パスメイカー用プラグイン
【Ver.】2.1
【作者】そぼろ

*************************************************************

移動速度に同期した回転運動のモーションファイルを生成します
タイヤの回転などに使えます

回転軸ベクトル、ボーン名、回転係数を指定して出力ボタンを押すと
保存ダイアログが表示されるので、名前を付けて保存してください

回転係数の目安として、直径10のタイヤの回転が12ぐらいです（360°/外周長さ）
