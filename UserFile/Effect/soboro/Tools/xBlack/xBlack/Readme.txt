【Ver.】1.1
【作者】そぼろ

*************************************************************

アクセサリ用のxファイルを真っ黒にするツールです
使い方は、xBlack.exeに変換したいxファイルをドラッグ＆ドロップするだけです
元ファイルと同じ場所に変換後のxファイルが作成されます

テクスチャのアルファチャンネルで透過しているところも
問答無用で塗りつぶされるのでご注意ください

厳密な構文解析をしているわけではなく、簡易な置換処理を行っているだけなので、
うまく変換できないxファイルもあるかもしれません
ソースを添付したので、自由に改造してもかまいません

姉妹ツールのpmdBlackと合わせてご活用ください

■更新履歴

Ver.1.1
日本語テクスチャを使ったファイルが変換失敗するバグ修正

■インストール

不要です
設定ファイルを作らないので、どこに置いても特に問題ないはず
アンインストールも、まとめて削除すればOKです

■動作環境

.NET Framework 2.0

（Vista64bitで動作確認）
