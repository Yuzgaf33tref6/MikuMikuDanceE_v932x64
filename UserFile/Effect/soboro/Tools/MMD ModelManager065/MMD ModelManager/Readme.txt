MMD特化ファイルブラウザ

【名称】MMD ModelManager
【Ver.】0.6β
【作者】そぼろ

*************************************************************


MMD関係のファイルに特化したファイルブラウザです。
モデルのプレビューやファイルのマイリスト管理ができます。
グラボを使わないのでPCに優しいです。

まだβ版ですので未実装の機能やバグも多いです。


・基本操作

ファイルを
　→クリック　　　　：選択してプレビュー表示
　→ダブルクリック　：開く
　→ホイールクリック：エクスプローラで開く
　→右クリック　　　：操作メニュー
　→Ctrl+右クリック
　　または右クリック長押し：エクスプローラの操作メニュー

フォルダを
　→クリック　　　　：選択
　→ダブルクリック　：開く
　→ホイールクリック：新しいタブで開く
　→右クリック　　　：操作メニュー
　→Ctrl+右クリック
　　または右クリック長押し：エクスプローラの操作メニュー

ファイルリスト余白を
　→ダブルクリック　：上のフォルダへ移動
　→ホイールクリック：エクスプローラで開く
　→右クリック　　　：操作メニュー

タブを
　→クリック　　　　：選択
　→ホイールクリック：閉じる
　→何もない場所を
　　ホイールクリック：直前に閉じたタブの復元

・お気に入り機能

　フォルダを登録して良く使うフォルダをすぐ開けるようにします。
　フォルダツリーの Bookmarks にあります。
　フォルダ右クリックから登録します。
　Delキーで削除、Ctrl＋上下キーで順序入れ替え。

・マイリスト機能

　ファイルを登録してフォルダ階層と無関係にファイルをまとめられます。
　マイリストはいくつも作れます。
　フォルダツリーの Mylists にあります。
　フォルダツリーので選択した状態でもう一度クリックすると名前変更できます。
　フォルダ右クリックから登録します。
　Delキーで削除。

・プレビュー
　いまのところ、モデルのワイヤーフレーム表示と画像のみ対応しています。

・オブジェクトツリー
　いまのところ、モデルのボーンを表示する機能のみ実装済みです。

・検索
　部分一致検索では、スペースで区切った複数の単語は基本的にAND検索になり、
　大文字の" OR "をはさむとOR検索、単語の先頭に"-"を付けるとマイナス検索になります。
　オプションで正規表現も使えますが、作者があまり詳しくないのでオマケ程度です。
　複数の検索結果を保持することはできません。

・簡易ランチャ
　ファイル一覧の下のプレビュー領域に外部ツールをドラッグ＆ドロップすると、
　ランチャに登録され、ファイルをそのツールで開くことができるようになります。
　PMDEditorなどを登録しておくと便利になるでしょう。
　ツールへのショートカットをドラッグ＆ドロップしてもOKです。

　ランチャ右クリックからランチャの編集を行うことができます。
　編集できる項目は以下の通りです。
　　・名前（選択したリストを再度クリック）
　　・パス
　　・コマンドライン引数（「%1」がプレビュー中のファイル名になります）
　　・フィルタ（セミコロン区切りで拡張子を指定します。
　　　　　　　　一致したファイルプレビュー時のみランチャに表示されるようになります）

・リンク機能
　歯車アイコンのメニューから、リンク先となる起動中のアプリケーション(MMDなど)を選択することで、
　リンクモードに入ることができます。
　リンクモードでは、ファイルをマウス左ボタン長押しすることで、選んだリンク先への
　ドラッグ＆ドロップを自動化することができます。


■必要環境

.NET Frameworks 4.0
ホイール付きマウス

※ Windows XPではレイアウトが崩れる可能性があります。


■設定の保存場所

設定はApplicationDataフォルダに保存されるため、
アップデートしても同じ設定が引き継がれます。

保存場所は・・・Cドライブを「SvSoftWorks」で検索したらそのうち出てきます。
