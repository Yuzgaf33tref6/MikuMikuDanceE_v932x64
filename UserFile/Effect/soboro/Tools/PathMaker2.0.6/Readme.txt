MMD PathMaker 2.0.6 β

作成：そぼろ


パスメイカーの進化版です。
まだ未完成ですので機能が完全ではありません。
また、エラー処理も十分でないので落ちやすいかもしれません。
バグ報告、要望などお待ちしてます。

以前の 1.x とはプロジェクトファイルおよびプラグインの
互換性がありませんのでご注意ください。


以下、目玉機能をごく簡単に。

・3D表示
　表示が3Dになりました。MMDとだいたい同じ操作です。

・速度編集
　専用の速度ビューを設け、速度もドラッグで値を変化させられます。

・レイヤー機能
　複数のパスを同時に扱えるようになりました。

・背景モデル読み込み
　まだPMDとPMXのみ対応ですが、ワイヤーフレームのモデルを表示できます。

・ランドマーク機能の強化
　通常のマーカーと同じように複数追加・削除したり、ドラッグで移動できます。

・再生機能の進化
　複数レイヤー同時再生、一時停止など機能強化されています。
　一時停止時は、速度ビューをドラッグでタイムバーを動かせます。
　また、非再生時に速度ビューをダブルクリックすると、
　その時間で一時停止された状態になります。

・モデルへの高さ揃え
　マーカーのY方向位置を自動的にモデルの面に合わせます。
　マーカーを面と面の境界あたりに配置していると、めり込みにくく効果的です。

・プレビューエフェクト
　MMEを利用してパスメイカーで編集中のパスをMMD上に表示します。



■必要環境

・Windows Vista/7/8
・.NET Framework 4
・それなりのCPUとメモリ
・ホイール付きマウス


■プレビューエフェクトの使い方

「表示」＞「プレビューエフェクト出力」をクリックし、
出力するエフェクトファイルの名前を指定します。
同名のXファイルが同時に作られるので、これをMMDで読み込み
描画順を最後にします。
Siでラインの太さを変更できます。

その後、パスが更新されると自動的にエフェクトも更新されます。
更新間隔は約１秒です。

パスメイカーを終了するとプレビューエフェクトの設定も解除されるので、
再度起動した際はもう一度プレビューエフェクト出力を行ってください。


■Tips

・マーカードラッグ時にX、Y(またはC)、Zキーを押すと移動方向を制限できます。
