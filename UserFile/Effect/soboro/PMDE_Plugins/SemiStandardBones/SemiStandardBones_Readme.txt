PMD形式において、標準とは言えないまでも、
広く使われているボーンを追加するPMDEditor用のプラグインです。

PMDEの"_plugin\User"フォルダに"SemiStandardBones.dll"を入れれば導入完了です
PMDEを起動すればメニューに登録されているはずです

ウェイト付けも自動的に行われますが、複雑な服装や装飾品を持ったモデルでは
上手く動作しない可能性があります。
名前の横に「*」マークのあるボーンがウェイト付けを行うボーンになります。

また、標準ではない複雑なボーン構造を持ったモデルでは
上手く動作しない可能性もあります。
MMD付属の初音ミクVer.1互換のボーン構造のモデルを推奨します。


■対応状況

PMDE 0.1.2.6より、PMDモードでの SemiStandardBones の正常動作を確認しました。

また、PMX専用の SemiStandardBonesPMX を同梱しました。
SemiStandardBonesPMX で作成されるボーンはPMDでは
再現不可能な構造を含むため、PMDには使えません。
PMDE 0.1.2.7 以前ではPMDEのバグにより不具合が出るので、
必ず0.1.2.7以降を使ってください。


■全選択ボタン

押すと、すべてのチェックボックスが選択されます。
すべて選択された状態でもう一度押すと、すべて解除されます。


■素材選択

素材選択ボタンを押すと素材リストが現れるので、
ウェイトを自動設定しては困る材質のチェックを外してください。


■注意事項

Win7において、セキリュティのためファイルがブロックされている可能性があります。
SemiStandardBones.dllを右クリックして"プロパティ"を選択し、
ブロックを解除してください。
