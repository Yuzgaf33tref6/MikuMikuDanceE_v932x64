ikOverray2

ikOverrayの機能を減らしたもの。
アニメっぽい画面外からの光を表現する。

使い方：
ikOverray2.x をmmdに入れる。
ikOverray2Controller.pmx (以下、コントローラ) を mmdに入れる。
PostDepth.x をmmdに入れる。
	これは後述する"厚み"の計算に必要。厚みが不要ならいれなくてもいい。

コントローラの"ターゲット"ボーンの向きで光の方向が変わる。
	(ターゲットボーンのローカル軸のY方向に仮の光がある)

コントローラのモーフで色などを調整。
	色：赤〜緑〜青〜赤で変化する。
	白：白っぽくなる
	明るさ：大きいほど薄くなる。
	短く：光の長さが短くなる
	丸い：光が丸くなる。0だと平坦。
	厚み：大きい値ほど、奥行きに応じて光の濃さが変わる。
