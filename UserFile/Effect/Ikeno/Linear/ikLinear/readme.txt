
リニアワークフローにするためのエフェクト

■ なにこれ？
パソコンが扱う色というのは人間の目に合わせて調整されていて、
光の計算を行うのには向いていません。
通常は、計算しやすいように調整 → 加工 → 出力用に戻す。とするか、
計算がおかしいのを無視してそのまま加工します。

複数のエフェクトを使う場合、
　計算しやすいように調整1 → 加工1 → 出力用に戻す1
　計算しやすいように調整2 → 加工2 → 出力用に戻す2
としている処理を、
　計算しやすいように調整 (ikLinearBegin)
　　→ 加工1
　　→ 加工2
　→ 出力用に戻す (ikLinearEnd)
のように改善します。


■ 対応しているエフェクト

AL4Polish
　ikPolish用に改造したAutoLuminous

AutoLuminousL
MotionBlurL
　※ AutoLuminous、MotionBlur は色の調整しないタイプのエフェクトですが、
　ikLinearがない場合は自分で色の調整を行うように改変しています。
　またMotionBlurLは、1以上の明るさに対応しました。いわゆるHDR対応というやつです。

ikBokeh v016以降
　v015以前は独自に色の調整を行っているので、ikLinearとの相性は悪いです。
ikSunshaft v006以降
ikPolishShader v016以降
　公開準備中。

未対応のポストエフェクトは、ikLinearBeginの前、
または ikLinearEndの後に配置してください。
