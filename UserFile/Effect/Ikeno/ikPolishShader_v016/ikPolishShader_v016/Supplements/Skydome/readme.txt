
とりあえずの背景が必要な場合の仮背景。

MMEのEnvMapタブで、skydome.xに対して、TEnvMap_emissive.fxを割り当ててください。

テクスチャの太陽の位置にライトをあわせる場合の設定：
赤:250, 緑:247, 青:235
X: 0.89, Y: -0.92: Z: 0.44
(※skydome.xを回転させない場合)


スカイドームのテクスチャは、sIBL Archive の Grand Canyon C を加工しています。
より大きいサイズのテクスチャが必要な場合は、以下からオリジナルの画像を
ダウンロードしてください。
http://www.hdrlabs.com/sibl/archive.html
