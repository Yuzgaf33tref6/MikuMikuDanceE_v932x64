Color.fx ver0.0.3
Color_Flat.fx ver0.0.3

モデルの色変更を行います．コントロールモデルの表情設定で全てのパラメータ変更が出来ます．
Color.fxは単色塗りつぶし,グレースケール,セピア色,ネガポジ変換が出来ます．
Color_Flat.fxは完全べた塗りでモデルを単色で塗りつぶします．シルエット描画になります．
舞力介入P氏のfull.fxを改変して作成しました．


・使用方法
(1)PMD,またはアクセサリにColor.fxまたはColor_Flat.fxを適用してください．
    GUIの場合｢MMEffect｣→｢エフェクト割当｣のメインタブより適用したいオブジェクトにColor.fxを割り当てる.
    ファイル名設定の場合：XXXXX.pmd から XXXXX[Color.fx].pmd に変更，または Color.fx を XXXXX.fx に変更してロード．
(2)ColorControl.pmdをMMDにロードします．
(3)モデル描画順序で適用したオブジェクトの直前にColorControl.pmdを置く．
(4)ColorControl.pmdの表情スライダのそれぞれの項目で以下の制御が可能です．
    目  ：赤,緑,青 → 塗りつぶしRGB色を指定します．
    ﾘｯﾌﾟ：色相,彩度,照度 → 塗りつぶしHSV色を指定します．
    まゆ：ブレンド → RGB色とHSV色の影響度決めて塗りつぶし色を決めます．
          エッジも → エッジも色変換の対象にします．
    その他：α値 → 塗りつぶし色をモデルの色と線形合成します．
            グレー,セピア,ネガ → それぞれグレースケール,セピア色,ネガポジ変換します．(Color_Flat.fxでは使用できません)
