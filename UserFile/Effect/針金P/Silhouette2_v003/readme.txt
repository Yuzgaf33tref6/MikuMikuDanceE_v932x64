Silhouette2.fx ver0.0.3
Silhouette2_MMM.fxm ver0.0.3

テクスチャ画像を用いてモデルのシルエット描画を行います。
以前作成したSilhouette.fxを見直してオフスクリーンレンダを使わずに使用できるように再構築しました。
その結果，描画にかかる負荷を大幅に軽減でき，モデルごとに個別の適用が楽に出来るようになりました。
シルエット映像には単色,テクスチャ,アニメGIF･APNGアニメが使用できます(Screen.bmpアニメは
使用できなくなりました)。マスク画像を用いてフェードイン・フェードアウトも行えます。
舞力介入P氏のfull.fxを改変して作成しました。


・使用方法
[MMD･MME版の場合]
(1)PMD,またはアクセサリにSilhouette2.fxを適用してください。
    GUIの場合｢MMEffect｣→｢エフェクト割当｣のメインタブより適用したいオブジェクトにSilhouette2.fxを割り当てる.
    ファイル名設定の場合：XXXXX.pmd から XXXXX[Silhouette2.fx].pmd に変更，または Silhouette2.fx を XXXXX.fx に変更してロード。
(2)Silhouette2.fxの先頭パラメータを適宜変更してください。
(3)SilhouetteControl.pmdをMMDにロードします。
(4)モデル描画順序で適用したオブジェクトの直前にSilhouetteControl.pmdを置く。
(5)SilhouetteControl.pmdのボーンと表情スライダのそれぞれの項目で以下の制御が可能です。
    ボーン：ｽｸﾛｰﾙ → XY移動でテクスチャのスクロール表示をします。
    目  ：単色赤,単色緑,単色青 → 単色の場合のRGB色を指定します。テクスチャ表示の場合は不要
    ﾘｯﾌﾟ：拡大,縮小 → テクスチャの拡大･縮小を行います。単色の場合は不要
    まゆ：加算合成 → シルエットとなる色を加算合成で描画します。
          エッジも → エッジもシルエット描画の対象にします。
          地面影も → 地面影もシルエット描画の対象にします。
    その他：透過度 → シルエット色をモデルの色と線形合成します。
            フェード → フェードの進行度，0で完全透過，1で完全マスク，よってフェードインは1→0，フェードアウトは0→1
            閾値 → 値が小さいとフェードの変化がシャープで大きいとマイルドになります。

[MikuMikuMoving版の場合]
(1)Silhouette2_MMM.fxmをMMMにロードしてください。
(2)MMMメニューの｢ファイル｣→｢エフェクト割当｣より適用したいモデルを選択して、Silhouette2_MMMを割り当てる。
(3)SilhouetteControl.pmdをMMMにロードします。あとはMMD･MME版と同じです。
