Ghost.fx ver0.0.5

モデルの半透明化･フェードで幽霊表現を行います。
半透明化でモデルの中身が透けて見えることはありません。
舞力介入P氏のMirror.fxを改変して作成しました。


・使用方法
(1)Ghost.xをMMDにロードします。MMMではGhost.fxを直接ロードします。
(2)以下のどちらかの方法でエフェクトファイルをそれぞれ設定します。

    ①GUI設定の場合
      ｢MMEffect｣→｢エフェクト割当｣の適用したいオブジェクトを選択して、
        Main       ：Ghost_Object.fxを適用するか非表示にする。
        GhostRT    ：標準描画(解除ボタンを押す)→ここで別のシェーダ系エフェクトを適用することも出来ます。
        MaskGhostRT：Ghost_Mask1.fxを適用。MMMではGhost_Mask1_MMM.fxmをロードして適用します。

    ②Ghost.fxの先頭パラメータで設定する場合
      Ghost.fxの先頭にある所定の箇所にモデルファイル名を記述。最大10体まで定義可能

      ｢MMEffect｣→｢エフェクト割当｣のメインタブよりGhost_Object.fxを適用するか非表示にする。

(3)必要に応じてGhost.fxの先頭パラメータでフェードの高さ範囲と閾値を指定してください。
(4)MMDのアクセサリパラメータで以下の変更を行いキーフレーム登録してください。
    Si：(0～1)フェードの進行度，フェードインは0→1，フェードアウトは1→0
    Tr：モデルの透過度
    X ：フェード開始相対高さ(Ghost.fxの先頭パラメータ＋この値が実際の指定高さになる)
    Y ：フェード終了相対高さ(Ghost.fxの先頭パラメータ＋この値が実際の指定高さになる)
      デフォルトでは下側からフェードで透明になる。開始･終了高さを入れ替える(例:X=20,Y=-20)と上側からのフェードになる。
    MMMではエフェクトプロパティに追加したUIコントロールより変更が可能です。
(5)Ghost.xの描画順序は出来るだけ最後の方に設定してください。


・注意点
Ghost_Object.fxを適用した場合,モデルにセルフシャドウ描画が行われます。その際,背景モデルにも半透明モデルの
影が落ちることになります。影を消したい場合はGhost_Object.fxの代わりに非表示選択するか、半透明モデルの
セルフシャドウをOFFにしてください。
