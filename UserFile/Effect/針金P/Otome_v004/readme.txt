乙女フィルター ver0.0.4

水玉とキラキラのアニメ演出に出てくる乙女チックなアレっぽいエフェクトです．一応ドラマ演出向けです．
カメラをぐりぐり回すような使い方には向かないかも・・
特定のオブジェクト周辺にかける場合(Otome.fx)と画面全体(Screen_Otome.fx,Post_Otome.fx：カメラ固定)を用意しました．
舞力介入P氏のlaughing_man.fx,FireParticleSystem.fxを改変して作成しました．


・Otome.fx使用方法
(1)Otome.xをMMDにロードしてください．
(2)描画順序は対象となるモデルの後にしてください．
(3)必要に応じてfxファイルの先頭パラメータを変更してください．
(4)MMDの通常のアクセサリパラメータ変更で位置,大きさ,透過度を設定してください．

・Screen_Otome.fx使用方法
(1)Screen_Otome.xをMMDにロードしてください．
(2)描画順序はできるだけ後ろの方にしてください．
(3)必要に応じてfxファイルの先頭パラメータを変更してください．
(4)MMDのアクセサリパラメータはTr(透過度)のみ使用できます．

・Post_Otome.fx使用方法
Post_Otome.xをMMDにロードしてください．使い方はScreen_Otome.fxと同じです．
こちらはポストエフェクトとして動作します．DOF等の他のポストエフェクトの影響を
受けずに描画することが出来ます．
描画順序は必ず他のポストエフェクトの後にしてください．


・MikuMikuMovingについて
このエフェクトはMikuMikuMovingにも対応しています。
Otome.fx,Screen_Otome.fx,Post_Otome.fxを直接MikuMikuMovingにロードしてご利用下さい。


同梱のBack.bmpは背景BMP,またはScreenTex.fx等でご使用ください．

このfxファイルはMME ver0.24以降でないと正しく動作しません。
