幻想少女大戦風ゲージエフェクト＆ダメージエフェクト
作：ビームマンP

某SRWっぽい同人ゲームっぽいゲージエフェクトです。ぽいぽい


使い方
幻想少女大戦風システム表示（左）.pmx
幻想少女大戦風システム表示（右）.pmx
ダメージ表示.pmx
をMMDに読み込みます

それぞれ
センターボーンのXY座標で表示座標調整、Z座標で表示サイズ調整を行います。

「幻想少女大戦風システム表示」は調節カテゴリ、
HPMAX、MPMAX、現在HP、現在MPの"X座標"を操作する事で表示数値を調節できます。

また、簡易HP、簡易MPモーフを使う事で現在HP、現在MPを減らすこともできます。

初期設定では現在HPが10000を超えると表示が？？？？？になります。
閾値はfxファイルを開き（幻想少女大戦風システム表示（左、もしくは右）.fx）

//体力が?????になる値
#define SECRET_HP 10000

↑こんな感じの場所の数値を変える事で変更できます



ダメージ表示.pmxはシステム表示と同じくセンターボーンで座標、サイズの操作
調節カテゴリ、ダメージボーンの"X座標"で表示する数値を調整
「進行」モーフでアニメーションを行います


おまけ　フォルダ内の「SRWスタイル」は　"MP"の表記が"EN"になっただけです　　　だけです！
