歪み軌跡・使い方

歪む軌跡です！
基本的にパラメータはfxファイルを直接編集します。

主なパラメータ

float4 LineColor = float4(1,0.1,0.1,1);
着色する色をRGBAで指定します。0,0,0,1で無着色、1,1,1,1で白です。

float DistPow = 0.25;
歪む力を設定します。0で歪みません。

float CutSpeed = 0;
軌跡を生成する最低速度を指定します。
ゆっくり動かすと軌跡が発生しないようにしたい時等に使用します。
尚、0の場合常に生成されます。

#define LINE_LENGTH 100
軌跡の長さです。

float lineSize = 0.1;
ラインの太さです。
MMD上のSi値でも操作が可能です。

float LineSpd = 0.0;
ラインを射出する速度です。
射出方向はモデルのZ方向です。
