LineSystemの剣軌跡用設定ファイル

使い方：
・ＭＭＥを入れる
・何か武器を持たせる（基本的に直刀を想定。鎌とかはなんかもうがんばれ）
・blade.xを読み込む
・持たせた方の手首にblade.xを設定する
・アクセサリ操作にて、Ryを90にする（角度合わせ）
・X(前後調整),Z（高さ調整）をずらして、blade.xの中心を剣軌跡を発生させたい座標に合わせる
・Siで剣軌跡の大きさを調整する
・blade.fx内の#define LINE_LENGTH 15←ここの数字を大きくしたり小さくしたりして、軌跡の長さを調整する
・テクスチャは、同名のテクスチャで上書きするか、blade.fxの
//テクスチャ名
texture Line_Tex
<
   string ResourceName = "blade.png";←ここで変更
>;
