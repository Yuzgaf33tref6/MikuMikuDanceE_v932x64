ビームマグナム風エフェクト
作：ビームマンP

使い方（MMD)
1:BeamMagnum.xを読み込む
2:出る
3:Siで大きさ、Trで太さを調節する
4:BeamMagnum.fxを開いてInnerSize、OuterSizeパラメータを調節する >= 2.0;とかなってるところ

使い方（MMM)
1:BeamMagnum.fxを読み込む
2:出る
3:拡大率で大きさ、透明度で太さを調節する
4:InnerSize、OuterSizeスライダでいい感じに調整する　MMM便利！
5:ただしバチバチは出ない。だってMMMが対応してないんだもの…

おまけ
BeamMagnum_Thunder(MMD専用）
バチバチ部分のみ。
使い方は上記と一緒　当然MMMでは動かない　しょぼん
